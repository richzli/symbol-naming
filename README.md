# symbol-naming

We use lots of symbols in code. However, names for these symbols are often long and cumbersome to say out loud. I propose a short one- to two-syllable name for each ASCII printable character.

### Extant

- `!`: bang
- `#`: sharp, pound, hash
- `&`: and
- `*`: star
- `+`: plus
- `,`: comma
- `-`: dash, *sub*
- `.`: dot
- `/`: slash
- `@`: at
- `^`: carat
- `` ` ``: tick
- `|`: bar
